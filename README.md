# `weekly-summary`

## 📝주간 작업 / 공부한 내용들의 정리 및 회고.

### - [`/report`](/report)

- 작업 내역을 보고서로 작성.
- 목표: 매주 작업한 내역을 확인 및 비교하여 발전된 작업을 위한 촉진제.

### - [`/research`](/research)

- 자료 조사 및 공부 / 연구한 내용을 정리하여 작성.
- 자기 자신이 읽는 목적이 아닌, 모두가 볼 수 있는 내용이 될 수 있도록 작성.
- 목표: 지식, 인사이트 함양 및 글 작성 기법 숙달.

### - [`/retrospect`](/retrospect)

- 생각을 정리하여 회고한 내용을 작성.
- 주제 없이 다양한 내용을 정리하여 작성.
- 목표: 글 작성 기법 숙달.
